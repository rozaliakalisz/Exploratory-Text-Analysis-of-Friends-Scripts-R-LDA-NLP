The project was developed as part of the Information Systems Design course at the Faculty of Economic Sciences, University of Warsaw. The goal was to create a tool for exploratory analysis of text data from the TV series Friends, using the R language and the Reproducible Research approach. The project includes word frequency analysis, topic modeling (LDA), result visualization, and analysis of word associations with the show's main characters.

Projekt został przygotowany w ramach zajęć Projektowanie Systemów Informatycznych na Wydziale Nauk Ekonomicznych Uniwersytetu Warszawskiego. Celem było stworzenie narzędzia do eksploracyjnej analizy danych tekstowych z serialu Friends, z wykorzystaniem języka R i podejścia Reproducible Research. Projekt obejmuje analizę częstości słów, modelowanie tematów (LDA), wizualizację wyników oraz analizę asocjacji słownych z głównymi postaciami serialu.

Authors/Autorzy: Maria Jastrzębska, Rozalia Kalisz
